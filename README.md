# Evaluación Quarkus: Microservicio Exchange

Este proyecto es un microservicio **Java** utilizando **Quarkus** para construir una API de tipos de cambio.

---

## 🚀 Configuración del Proyecto

### 1️⃣ Clonar el repositorio

```bash
git clone https://github.com/gpalacios26/exchange-quarkus.git
cd exchange-quarkus
```

### 2️⃣ Ejecutar la aplicación

Con **Maven**:

```bash
mvn clean install -DskipTests

mvn quarkus:dev -DskipTests  
```

---

## 🌐 Endpoints de la API

Utilizar la colección `Exchange-Quarkus.postman_collection.json` que se encuentra dentro de la carpeta postman.
